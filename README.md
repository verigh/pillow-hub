# Pillow Hub
Script for Pillow Fight Simulator

Current features include:
- Nice GUI (Arrayfield [Rayfield modded])
- Range Kill
- Spawn/Pickup ladder anywhere
- Transform any Player into Karen (Karenville only)
- Vote for any map
